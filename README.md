### Project Title
Bikeshare.py project

### Description
This project allows a user to investigate statistics concerning bikeshare data in the cities of
Chicago, New York and Washington. It provides timing data to demonstrate the power of numpy and
pandas modules as well as statistical analysis.

### Files used
bikeshare.py - python script file
chicago.csv - file is ignored in repository but needed for data
new_york_city.csv - file is ignored in repository but needed for data
washington.csv - file is ignored in repository but needed for data
.gitignore - standard file, includes the csv data files referenced
